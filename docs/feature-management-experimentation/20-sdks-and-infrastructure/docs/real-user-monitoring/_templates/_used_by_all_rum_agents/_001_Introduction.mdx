This guide provides detailed information about FME Real User Monitoring (RUM) Agent for {props.rumAgentType}.

FME {props.rumAgentType} RUM Agent collects events about your users' experience when they use your application and sends this information to FME services. This allows you to measure and analyze the impact of feature flag changes on performance metrics.
