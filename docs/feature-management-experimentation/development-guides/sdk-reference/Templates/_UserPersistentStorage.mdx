## User Persistent Storage

A custom storage adapter that allows the SDK the persist values for users in active experiments.
In otherwords, allowing you to run experiments with sticky bucketing.
You can provide a persistent storage adapter via [StatsigOptions.UserPersistentStorage](#statsig-options).

You can read more about the concept [here](/client/concepts/persistent_assignment).
