Input required notifications alert users when their input is needed to proceed with the release.

## When Input is Required

### During Execution
Input needed during execution:
- **Manual activities**: waiting for a user to record inputs or provide sign-off
- **On hold / input required**: an activity is blocked and requires action to proceed
- **Fix required**: an issue needs to be fixed before retrying or continuing

## Best Practices

### Clear Instructions
Provide clear instructions:
- **What's Needed**: What input is needed
- **Why Needed**: Why it's needed
- **How to Provide**: How to provide

### Timely Notifications
Send timely notifications:
- **Immediate**: Send immediately
- **Reminders**: Send reminders
- **Escalation**: Escalate if needed
- **Clear Urgency**: Indicate urgency
