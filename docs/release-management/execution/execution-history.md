The activity execution history shows the status and details of activities during release execution.

## Activity Execution History

During release execution, you can view the activity execution history which shows:
- **Activity Status**: Current status of each activity
- **Errors**: Any errors with respect to particular pipelines or activities
- **Execution Details**: What happened during activity execution

## Viewing Execution History

When monitoring a running release, you can:
- See which activities have executed
- View activity status and any errors
- See which activities are waiting for input or approval
- Understand what's happening with each release
