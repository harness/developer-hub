This topic describes the method to connect your chaos infrastructure to the control plane on [app.harness.io](https://app.harness.io) or Self-Managed Platform (SMP) via proxy when using a [dedicated chaos infrastructure](../infrastructure/types/legacy-infra).

To configure proxy settings for the [Harness Delegate Driven Chaos Runner](../infrastructure/types/ddcr/proxy-support#hnp-configuration-for-delegate-driven-chaos-runner-ddcr), go to the proxy support documentation.
