This topic describes the method to connect your chaos infrastructure to the control plane on [app.harness.io](https://app.harness.io) or Self-Managed Platform (SMP) via proxy when using a [dedicated chaos infrastructure](../infrastructure).

To configure proxy settings for the [Harness Delegate Driven Chaos Runner](../infrastructure/kubernetes/network-config#install-harness-network-proxy-hnp), go to the proxy support documentation.
