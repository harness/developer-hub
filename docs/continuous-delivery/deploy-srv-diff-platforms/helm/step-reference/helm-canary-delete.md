This topic describes the configuration and settings for the Helm Canary Delete step.

## Name

The name for the delete step.

## Timeout

Enter a timeout for this step. You can use: 
- `w` for weeks
- `d` for days
- `h` for hours
- `m` for minutes
- `s` for seconds
- `ms` for milliseconds
