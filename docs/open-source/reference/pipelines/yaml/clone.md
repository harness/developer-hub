# Clone

Clone defines the clone behavior.

## Properties

* __depth__ - _number_<br/>
  The clone depth.

* __disabled__ - _bool_<br/>
  The clone is disabled.

* __insecure__ - _bool_<br/>
  The ssl verification is disabled.

* __trace__ - _bool_<br/>
  The clone tracing is enabled.
